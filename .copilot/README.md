# Copilot Skills for TimeUp

This directory contains reusable skills (workflows) that GitHub Copilot can execute when working on the TimeUp project.

## Available Skills

### Development Workflows

- **add-feature.skill.md** - Add new features following architecture patterns
- **debug.skill.md** - Debug common issues and problems
- **test.skill.md** - Run tests and check coverage

### Deployment & Testing

- **serve-locally.skill.md** - Start local development server
- **test-in-trello.skill.md** - Test Power-Up in Trello
- **deploy.skill.md** - Deploy to GitHub Pages

## How to Use

Ask GitHub Copilot to execute a skill:

- "Use the add-feature skill to create a new timer pause function"
- "Run the debug skill to help me fix the badge not updating"
- "Execute test skill to verify my changes"

Copilot will follow the step-by-step workflows defined in these skill files.

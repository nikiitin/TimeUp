Privacy Policy

Last updated: January 27, 2026

Overview

TimeUp is a Trello Power-Up that provides time tracking functionality within Trello boards. This policy describes how TimeUp handles user data.

Data Collection

TimeUp does not collect, store, or transmit any personal data. The Power-Up operates entirely within your browser and uses only Trello's built-in storage API.

Authorization & Trello API Access

To provide accurate time estimates, TimeUp requests Read Access to your Trello boards via the Trello REST API. This authorization is strictly used to:

  • Fetch detailed checklist items (which are sometimes incomplete via standard Power-Up limitations)
  • Calculate progress based on checklist completion

This access is read-only for board data. We do not modify your boards or cards via the API (except for storing timer data, which uses the standard Power-Up storage).

Data Storage

All time tracking data (timer states, time entries, durations) is stored directly within Trello using their Power-Up storage API. TimeUp does not maintain its own database.

Personal Information

TimeUp does not access or store personal information like names or email addresses on any external server. All data remains within the Trello ecosystem.

Data Deletion

To remove all TimeUp data from a card, simply delete the time entries within the Power-Up interface. To remove all data from a board, disable the TimeUp Power-Up from that board.

Changes to This Policy

Any changes to this privacy policy will be reflected on this page with an updated revision date.

Contact

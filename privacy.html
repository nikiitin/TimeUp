Privacy Policy

Last updated: January 29, 2026

Overview

TimeUp is a Trello Power-Up that provides time tracking functionality within Trello boards. This policy describes how TimeUp handles data.

No External Data Collection

TimeUp does not collect, transmit, or store any data on external servers. The Power-Up operates entirely within your browser and uses only Trello's built-in storage API. There are no analytics, tracking cookies, or third-party services.

Data Storage Within Trello

All data is stored using Trello's Power-Up storage system, which keeps data within your Trello workspace. The following information is stored:

Card-Level Data (Shared with all card members)

  • Time entries: Start time, end time, duration, and optional descriptions
  • Timer state: Whether a timer is currently running, paused, or idle
  • Time estimates: Manual or calculated estimates for tasks
  • Checklist item timers: Individual timer states for checklist items

Board-Level Settings (Shared across board)

  • Hourly rate: Optional billing rate for reports
  • Currency: Display currency preference
  • Categories: Custom time entry categories

User-Level Preferences (Private to you)

  • Display format: Show seconds, 12/24-hour format
  • Behavior settings: Auto-start preferences

Trello API Access (Optional)

TimeUp requests optional Read-Only Access to your Trello boards via the Trello REST API. This authorization is used exclusively to:

  • Fetch complete checklist item details (standard Power-Up API has limitations)
  • Calculate progress based on checklist completion

You can decline this authorization and TimeUp will continue to work with basic checklist functionality. This access is read-only and does not modify your boards or cards.

Personal Information

TimeUp does not access, store, or process any personal identifying information such as names, email addresses, or contact details. All stored data consists solely of timestamps, durations, and user-provided task descriptions.

Data Deletion

To remove all TimeUp data from a card, simply delete the time entries within the Power-Up interface. To remove all data from a board, disable the TimeUp Power-Up from that board.

Changes to This Policy
